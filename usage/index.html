    使い方¶

    knowledgevba は 3 つの Excel ブックを目的別に使い分けます。

    ブック こんなときに開く
    登録修正.xlsm ナレッジを新しく書き留めたい / 既存ナレッジを編集・削除したい
    検索.xlsm 過去のナレッジを探したい / 詳細を見たい
    管理.xlsm フォーマットを増やしたい / 保存先フォルダを変えたい / ログを見たい

    各ブックは独立しており、3 人で並行して使うこともできます。データ本体（.txt ファイル）は data\ フォルダに置かれ、3 ブックすべてが同じフォルダを参照します。

    用語¶

    • フォーマット ― ナレッジの「型」のことです。例: 「障害対応」「作業手順」など。フィールド名・必須/任意・データ型をフォーマットごとに決めます
    • ナレッジ ― フォーマットに沿って入力した 1 件分のレコード。data\<FormatID>-NNNN.txt の形で保存されます（例: FAULT-0001.txt）
    • ナレッジ番号 ― <FormatID>-NNNN の形式の通し番号。登録時に自動採番されます

    1. ナレッジを登録する（登録修正.xlsm）¶

    1.1 新規ナレッジを登録する¶

    1. 登録修正.xlsm をダブルクリックで開く
    2. 「コンテンツの有効化」をクリック
    3. 起動時に表示される 「ナレッジ登録」シート（M-05） が登録画面です
    4. [フォーマット] ドロップダウンから、登録したい型を 1 つ選択
    5. フォーマットに応じたフィールドが画面上に展開されるので、内容を入力
      • 必須項目には印が付いています。空のままだと登録できません
      • 日付フィールドは YYYY/MM/DD の形式で入力
      • 複数行の説明は、セル内で Alt+Enter で改行できます
    6. [登録] ボタンをクリック
    7. 完了メッセージにナレッジ番号（例: FAULT-0001）が表示されたら登録成功
      • 同時に別の利用者が同じ番号で登録しようとした場合は、自動的に次の番号でリトライされます

    1.2 ナレッジを修正する¶

    1. 画面下部にある [ナレッジ修正] ボタンをクリック、または「ナレッジ修正」シート（M-06）タブを開く
    2. [ナレッジ番号] 欄に修正したいナレッジの番号（例: FAULT-0001）を入力
    3. [読込] ボタンをクリック → 該当ナレッジの内容がフィールドに展開される
    4. 修正したい箇所を編集
    5. [更新] ボタンをクリック → 修正が保存される
      • 既存の内容は backup\ フォルダに自動でバックアップされます

    1.3 ナレッジを削除する¶

    1. 上記 1.2 の手順 1～3 でナレッジを読み込む
    2. [削除] ボタンをクリック
    3. 確認メッセージで [はい] を選択
    4. ナレッジ本体は data\ から削除され、backup\ にバックアップが残ります

    2. ナレッジを検索する（検索.xlsm）¶

    2.1 キーワードで検索する¶

    1. 検索.xlsm をダブルクリックで開く
    2. 「コンテンツの有効化」をクリック
    3. 起動時に表示される 「検索」シート（M-08） が検索画面です
    4. [フォーマット] で、対象を絞り込みたいフォーマットを選択（「すべて」も選べます）
    5. [キーワード] 欄に検索したい語句を入力
      • 複数のキーワードはスペース区切り
      • 空欄のまま検索すると、フォーマット指定範囲のすべてのナレッジが結果に出ます
    6. [検索] ボタンをクリック
    7. 検索結果の一覧が下に表示されます
      • タイトル一致・本文一致を組み合わせたスコア順で並びます

    2.2 検索結果から詳細を表示する¶

    1. 一覧の中で見たい行を ダブルクリック
    2. 「ナレッジ詳細」シート（M-09） が開き、全フィールドが表示されます
    3. 戻るには上の [検索に戻る] ボタンをクリック

    3. フォーマットを管理する（管理.xlsm）¶

    3.1 フォーマット一覧を見る¶

    1. 管理.xlsm をダブルクリックで開く
    2. 「コンテンツの有効化」をクリック
    3. 起動時に表示される 「フォーマット一覧」シート（M-02） に、登録済みのフォーマットが並びます
      • 各行に [編集] [プレビュー] [削除] のボタンが付いています

    3.2 新しいフォーマットを作る¶

    1. M-02 上部の [新規作成] ボタンをクリック
    2. 「フォーマット設計」シート（M-03） が開く
    3. 各項目を入力
      • フォーマット ID ― 半角英大文字の短い識別子（例: FAULT、SAGYO）。ナレッジ番号の頭にも使われます
      • フォーマット名 ― 表示用の名前（例: 「障害対応」）
      • フィールド一覧 ― 1 行 1 フィールドで、フィールド名・型・必須/任意 を入力
        • 型は「文字列」「数値」「日付」「複数行」から選択
    4. [保存] ボタンをクリック → format\<フォーマット ID>.txt が生成されます

    3.3 フォーマットをプレビューする¶

    1. M-02 のフォーマット行で [プレビュー] ボタンをクリック
    2. 「フォーマット プレビュー」シート（M-04） に、登録画面でどう見えるかが表示されます

    4. 格納先フォルダを設定する（管理.xlsm）¶

    4.1 4 つのフォルダパスを変更する¶

    1. 管理.xlsm を開く
    2. 「格納先設定」シート（M-10） タブをクリック
    3. 4 つのフォルダ（data / format / ui / backup）の現在のパスが行ごとに表示されます
    4. 変更したいフォルダの左にあるチェックボックスにチェックを入れる（複数同時に選択可）
    5. [格納先を選ぶ] ボタンをクリック → フォルダ選択ダイアログが開く
    6. 新しいフォルダを選んで [OK] → 選択されたパスがチェック済みフォルダに反映される
    7. [保存] ボタンをクリック → 設定が確定し、次回起動から新しいパスを参照します

    保存先を変更した場合

    フォルダパスを変更しても、既存のファイル（ナレッジ・フォーマット・画面定義・バックアップ）は 自動移動しません。エクスプローラで手動コピーしてから保存してください。

    5. フォーマット変更チェック（管理.xlsm）¶

    フォーマットの定義（フィールド構成）を後から変更したとき、既に登録済みのナレッジとフォーマットの整合性をチェックします。

    1. 管理.xlsm を開く
    2. 「フォーマット変更チェック」シート（M-12） タブをクリック
    3. [対象フォーマット] ドロップダウンから、チェックしたいフォーマットを選択
    4. [チェック] ボタンをクリック
    5. 該当フォーマットの全ナレッジを走査し、不整合（フィールド過不足・型ミスマッチ等）が一覧表示されます
      • 結果に何も出ない場合は整合性が取れています
      • 不整合がある場合は、登録修正.xlsm のナレッジ修正画面で個別に修正してください

    6. ログを見る（管理.xlsm）¶

    1. 管理.xlsm を開く
    2. 「ログ」シート（M-14） タブをクリック
    3. 登録・修正・削除・検索などの操作履歴が日時順に並びます
      • 「いつ・どの操作・どのナレッジ番号・成功/失敗」が見られます

    7. システム設定（管理.xlsm）¶

    1. 管理.xlsm を開く
    2. 「システム設定」シート（M-11） タブをクリック
    3. 表示色・既定フォーマット等の全体設定を変更できます
    4. 変更後は Excel を再起動すると反映されます
