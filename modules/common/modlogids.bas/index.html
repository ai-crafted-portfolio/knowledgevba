    modLogIds.bas¶

    配置先: 共通モジュール（3 ブック共通） 種類: 標準モジュール 更新日: 2026-06-03 23:22

    保存方法¶

    下のコードをメモ帳に貼り付け、[名前を付けて保存] で次のように保存してください。

    • 場所: C:\KnowledgeMgr\installer\vba_modules\common\\
    • ファイル名: modLogIds.bas
    • ファイルの種類: すべてのファイル
    • 文字コード: ANSI（Shift-JIS）

    メモ帳の文字コードを ANSI にしないと、VBA の日本語が文字化けして動かなくなります。 UTF-8 で保存すると VBA Import 時に日本語が文字化けして動かなくなります。 改行コードは CRLF（Windows 標準）のままで OK です。
