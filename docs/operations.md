# 操作手順

ナレッジ管理ツール v2.1 の日常的な操作手順です。作業の内容に応じて、対応するブックを開いて使います。

| やりたいこと | 開くブック |
|---|---|
| ナレッジを探す・読む | 検索.xlsm |
| ナレッジを新しく登録する・直す | 登録修正.xlsm |
| フォーマットを設計する・各種設定をする | 管理.xlsm |

画面の切り替えは、ブック下部の Excel のシートタブで行います。

## 1. ナレッジを探す（検索.xlsm）

### 1.1 検索する（M-08 ナレッジ検索）

検索.xlsm を開くと M-08 ナレッジ検索画面が表示されます。次の 8 つの条件を、必要なものだけ入力します。

1. キーワード
2. 対象フィールドの絞り込み
3. フォーマット種別
4. 期間（開始）
5. 期間（終了）
6. カテゴリ
7. 担当者
8. ナレッジ番号の直接指定

キーワードを半角スペースで区切ると、すべての語を含むナレッジが対象になります。条件を入力したら検索ボタンを押します。検索結果は 1 行 1 件で、ナレッジ番号・フォーマット ID・件名・更新日時・一致したフィールド名が並びます。表示は最大 100 件で、超えた場合は警告が出ます。

ナレッジ番号がわかっている場合は、条件 8 にその番号を入力して検索すると 1 件を直接呼び出せます。

### 1.2 一覧で眺める（M-07 ナレッジ一覧）

M-07 ナレッジ一覧では、登録済みナレッジを一覧で確認できます。フォーマットや期間で表示を絞り込めます。

### 1.3 詳細を読む（M-09 ナレッジ表示）

検索結果や一覧からナレッジを選ぶと、M-09 ナレッジ表示画面で詳細を確認できます。M-09 は閲覧専用で、ここから内容を書き換えることはできません。修正したい場合は登録修正.xlsm の M-06 を使います。

## 2. ナレッジを登録・修正する（登録修正.xlsm）

### 2.1 新しく登録する（M-05 ナレッジ登録）

登録修正.xlsm を開くと M-05 ナレッジ登録画面が表示されます。

1. フォーマットを選びます。選んだフォーマットの入力項目が画面に表示されます。
2. 各項目に内容を入力します。必須項目には印が付いています。
3. 保存ボタンを押します。ナレッジ番号（`<フォーマット ID>-<4 桁連番>`）が自動で採番され、`data\` フォルダにファイルとして保存されます。

絵文字やサロゲートペアを含む文字は保存できません。保存時にエラーが表示されたら、その文字を取り除いてください。

### 2.2 既存のナレッジを直す（M-06 ナレッジ修正）

1. M-06 ナレッジ修正画面で、修正したいナレッジの番号を入力して読み込みます。
2. 内容を編集します。
3. 保存します。読み込んでから保存するまでの間に、ほかの操作で同じナレッジが更新されていた場合は「上書き / 再読込 / 取消」を選ぶメッセージが表示されます。状況に応じて選んでください。

## 3. フォーマットを管理する（管理.xlsm）

### 3.1 フォーマット一覧（M-02）

管理.xlsm を開くと M-02 フォーマット一覧が表示されます。登録済みのフォーマットが一覧で並び、行を選んで新規作成・編集・プレビュー・削除を選べます。

### 3.2 フォーマットを設計する（M-03 フォーマット設計）

M-03 フォーマット設計では、フォーマット ID・名称・ナレッジ番号の採番パターンを決め、入力項目（フィールド）を設計します。各フィールドには名前・型（テキスト / 複数行テキスト / 数値 / 日付 / ドロップダウン）・必須かどうかを設定します。

フォーマットを保存すると、そのフォーマットを使う既存ナレッジへ変更が自動で反映されます。反映の前に自動でバックアップが取られ、確認のメッセージが表示されます。

フォーマットを削除しようとしたとき、そのフォーマットに紐づくナレッジが 1 件でも残っていれば削除は中止され、件数が表示されます。

### 3.3 表示を確認する（M-04 フォーマットプレビュー）

M-04 フォーマットプレビューでは、設計したフォーマットが入力画面でどう見えるかを確認できます。プレビューであり、ここでナレッジを保存することはありません。

## 4. データの場所と動作を設定する（管理.xlsm）

### 4.1 格納先設定（M-10）

M-10 格納先設定では、ナレッジ・フォーマット・画面定義・バックアップの格納先フォルダを確認・設定します。

### 4.2 設定（M-11）

M-11 設定では、操作ログの詳細度などの動作設定を変更できます。設定値の意味と変更方法は[設定値の変更](settings.md)を参照してください。

### 4.3 ファイル形式設定（M-13）

M-13 ファイル形式設定では、出力ファイルの形式に関する 15 項目を設定します。

### 4.4 フィールド反映（M-12）

フォーマットの項目を後から変更したとき、その変更を既存ナレッジへまとめて反映する画面が M-12 フィールド反映です。対象のフォーマットを選んで反映を実行します。データ消失のおそれがある変更では、反映前にバックアップが取られます。

## 5. 操作ログを確認する（M-14 操作ログ）

M-14 操作ログでは、管理.xlsm 自身の操作ログを表示・書き出しできます。M-14 には専用の絞り込み・並べ替え機能はありません。必要に応じて Excel 標準の機能を使ってください。

M-14 からはバックアップの管理もできます。保持期間（既定 90 日）を過ぎたバックアップは起動時に自動で削除されますが、M-14 から手動でまとめて削除することもできます。

## 6. 困ったときに確認すること

- **検索しても 0 件になる** — 格納先設定（M-10）の `data_dir` が、実際にナレッジファイルがあるフォルダを指しているか確認します。
- **登録時にフォーマットが選べない** — `formats\` フォルダにフォーマット定義ファイルがあるか、格納先設定の `format_dir` が正しいかを確認します。
- **画面が崩れて表示される** — 画面レイアウト定義ファイル（`ui\<ブック>\<画面>.txt`）が配置されているかを確認します。
- **ボタンを押しても反応しない** — ボタンに正しいマクロが割り当てられているか（ボタンを右クリック →「マクロの登録」）を確認します。配置手順は[導入手順](setup.md)を参照してください。
- **詳しい記録を見たい** — 設定（M-11）で `debugLevel` を `INFO` や `DEBUG` に上げると、LOG シートに詳しい記録が残ります。
