# ユーザマニュアル — 全体像

エンドユーザ向けの操作説明書です。セットアップから日々の操作まで、検索ブック・管理ブックの使い方をひととおりまとめています。

## はじめに

- 本ツールは、社内のナレッジ（障害対応・作業手順など）を決まった型で蓄積し、すばやく探し出すための Excel ツールです。
- 型（フォーマット）に沿って入力するため、誰が書いても項目がそろい、後から検索しやすくなります。
- ふだんお使いの Excel（Windows 版デスクトップ）だけで動作し、特別なサーバや専用ソフトは不要です。
- 本ツールは「検索ブック」「管理ブック」の 2 つの Excel ファイル（.xlsm）で構成されます。

!!! warning "動作環境"
    Excel for Mac やブラウザ版 Excel では動作しません。Windows 版のデスクトップ Excel をお使いください。

## 2 つのブックを使い分ける

### 🔍 検索ブック（検索.xlsm）

- ナレッジを探す（キーワード検索）
- 見つけた内容の詳細を見る
- 新しいナレッジを書き留めて登録する
- 操作の履歴（ログ）を残す

### 🛠 管理ブック（管理.xlsm）

- ナレッジの型（フォーマット）を作る・直す
- 型の変更が既存データに影響しないか点検する
- データの保存先フォルダを設定する
- 全体の設定変更・ログ確認を行う

!!! tip "使い分け"
    2 つのブックは独立して動きます。検索担当と管理担当が別々に、同時に使うこともできます。

## データの保存場所

ナレッジ本体や設定は、既定で `C:\KnowledgeMgr\` フォルダにまとめて保存されます。

| フォルダ | 内容 |
|---|---|
| `data` | 登録したナレッジ本体（1 件 1 ファイル） |
| `formats` | ナレッジの型の定義 |
| `ui` | 各画面のレイアウト定義 |
| `backup` | 修正・削除時の自動バックアップ |
| `log` | 操作履歴のログ |
| `config` | 動作設定ファイル |

!!! tip "覚えておくと便利"
    2 つのブックは同じ `C:\KnowledgeMgr\` を参照します。フォルダごと移動・コピーすればデータを引き継げます。
