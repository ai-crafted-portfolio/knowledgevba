# 検索ブックの使い方

ナレッジを探す・見る・登録するためのブックです。

## 検索ブックを開く

1. **検索.xlsm をダブルクリックする** — `C:\KnowledgeMgr\` にある検索ブックを開きます。
2. **マクロを有効にする** — 黄色い警告バーが出たら『コンテンツの有効化』をクリックします。
3. **検索画面が開く** — 起動と同時に『ナレッジ検索』画面が表示されます。ここから操作を始めます。

!!! tip "画面の構成"
    検索ブックには『ナレッジ検索』『ナレッジ登録』『ログ』の画面があり、下のタブで切り替えられます。

## キーワードでナレッジを探す

1. **フォーマット（型）で絞り込む** — 対象を絞りたいときは型を選びます。『すべて』を選ぶと全件が対象になります。
2. **キーワードを入れる** — 探したい語句を入力します。複数の語はスペースで区切ると、より絞り込めます。
3. **『検索』ボタンを押す** — 結果の一覧が画面の下に表示されます。
4. **並び順を確認する** — タイトル一致・本文一致を組み合わせた、関連度の高い順に並びます。

!!! tip "ヒント"
    キーワードを空欄のまま検索すると、選んだ型のナレッジがすべて一覧に出ます。

## 検索結果から詳細を見る

1. **見たい行をダブルクリックする** — 一覧の中から見たいナレッジの行をダブルクリックします。
2. **詳細画面で全項目を確認する** — そのナレッジのすべての項目が表示されます。
3. **検索に戻る** — 上部の『検索に戻る』ボタンで、いつでも一覧に戻れます。

!!! tip "編集したいとき"
    詳細画面では内容の確認ができます。修正したい場合は、登録画面からナレッジ番号で読み込んで編集します。

## 新しいナレッジを登録する

1. **『ナレッジ登録』画面を開く** — 検索ブックの下のタブから『ナレッジ登録』を選びます。
2. **フォーマット（型）を選ぶ** — 登録したい型をドロップダウンから 1 つ選びます。
3. **展開された項目に入力する** — 型に応じた入力欄が画面に並ぶので、内容を入力します。
4. **『登録』ボタンを押す** — 完了メッセージにナレッジ番号（例：FAULT-0001）が出れば登録成功です。

!!! tip "自動採番"
    番号は自動で採番されます。同じ瞬間に他の人が登録しても、自動で次の番号になり重複しません。

登録の流れをくわしく知りたいときは [登録・編集の流れ](register.md) を参照してください。
