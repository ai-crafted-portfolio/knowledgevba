# 登録・編集の流れ

入力から保存・反映までを、ひとつながりで説明します。

## 登録のひとつながりの流れ

1. **型を選ぶ** — 登録画面でフォーマット（型）を選ぶと、その型の入力欄が画面に展開されます。
2. **内容を入力する** — 展開された各項目に入力します（必須項目は空のままだと登録できません）。
3. **登録する** — 『登録』ボタンを押すと、内容が 1 件のナレッジとして保存されます。
4. **結果を確認する** — ナレッジ番号が表示され、以降は検索で見つけられるようになります。

!!! tip "保存の仕組み"
    保存されたナレッジは `data` フォルダに 1 件 1 ファイルで残ります。

## 入力のコツ

- 必須の項目には印が付いています。空のままだと登録できないので、まず必須から埋めます。
- 日付の項目は『2026/06/17』のように、年／月／日をスラッシュで区切って入力します。
- 長い説明を改行したいときは、セルの中で Alt キーを押しながら Enter キーを押します。
- 型に合わない値（数値欄に文字など）は登録時に知らせてくれるので、表示に従って直します。

!!! tip "読み手に親切な書き方"
    あとから見返す人のために、タイトルは具体的に。検索のヒット率が上がります。

## 既存ナレッジを直す・消す

1. **ナレッジ番号で読み込む** — 登録画面でナレッジ番号（例：FAULT-0001）を入れ、『読込』を押します。
2. **内容を編集する** — 読み込んだ項目を直し、『更新』を押すと修正が保存されます。
3. **不要なら削除する** — 削除する場合は『削除』を押し、確認に『はい』で消します。

!!! tip "安心ポイント"
    修正・削除のときは、変更前の内容が `backup` フォルダに自動で控えとして残ります。
