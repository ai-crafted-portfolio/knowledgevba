# 用語集・付録

## 用語集

### フォーマット（型）

ナレッジの「型」。項目名・必須／任意・データ型を型ごとに決めます（例：障害対応、作業手順）。

### ナレッジ

型に沿って入力した 1 件分の記録。1 件 1 ファイルで `data` フォルダに保存されます。

### ナレッジ番号

『型の ID ＋通し番号』の形（例：FAULT-0001）。登録時に自動で付きます。

### バックアップ

修正・削除のときに自動で残る変更前の控え。`backup` フォルダに保存されます。

## データの場所とお問い合わせ

すべてのデータは既定で `C:\KnowledgeMgr\` 配下に保存されます。

| 内容 | 場所 |
|---|---|
| ナレッジ本体 | `data` フォルダ |
| 型の定義 | `formats` フォルダ |
| 自動バックアップ | `backup` フォルダ |
| 操作ログ | `log` フォルダ |

- PC を移し替えるときは、この `C:\KnowledgeMgr\` フォルダごとコピーすればデータを引き継げます。
- 操作で困ったときは、本書の [困ったとき](faq.md) を確認のうえ、社内の管理担当者へご連絡ください。
