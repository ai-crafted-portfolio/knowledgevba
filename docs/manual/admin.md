# 管理ブックの使い方

型（フォーマット）の管理、保存先の設定、全体設定とログ確認を行うブックです。

## 管理ブックを開いて一覧を見る

1. **管理.xlsm をダブルクリックする** — `C:\KnowledgeMgr\` にある管理ブックを開きます。
2. **マクロを有効にする** — 黄色い警告バーが出たら『コンテンツの有効化』をクリックします。
3. **フォーマット一覧が開く** — 起動と同時に、登録済みの型（フォーマット）が一覧で表示されます。
4. **各行のボタンを使う** — 各行に『編集』『プレビュー』『削除』のボタンが並びます。

!!! tip "画面の構成"
    管理ブックには、一覧・設計・変更チェック・格納先設定・設定・ログの画面があり、下のタブで切り替えます。

## 新しいフォーマット（型）を作る

1. **『新規作成』ボタンを押す** — 一覧の上部にある『新規作成』を押すと、設計画面が開きます。
2. **フォーマット ID と名前を入れる** — ID は半角英大文字の短い記号（例：FAULT）。名前は表示用（例：障害対応）です。
3. **項目（フィールド）を 1 行ずつ入れる** — 項目名・型（文字列／数値／日付／複数行）・必須か任意かを入力します。
4. **『保存』ボタンを押す** — 保存すると新しい型の定義ファイルが作られ、登録・検索で使えるようになります。

!!! tip "ID の役割"
    フォーマット ID は、登録したナレッジ番号の先頭にも使われます（例：FAULT-0001）。

くわしい設計の手順は [フォーマット設計の流れ](format.md) を参照してください。

## フォーマットを確かめて使う

- 一覧の各行の『プレビュー』ボタンを押すと、その型が登録画面でどう見えるかを確認できます。
- 『編集』ボタンで、既存の型の項目をあとから増減・修正できます。
- 保存した型は、検索ブックの『ナレッジ登録』画面のフォーマット選択にすぐ表示されます。
- 型を選んでから登録すれば、項目がそろった状態でナレッジを蓄積できます。

!!! tip "反映のタイミング"
    型を増やしたあとは、検索ブックを開き直すと選択肢に反映されます。

## フォーマット変更チェック

型の項目を後から変えたとき、すでに登録済みのナレッジと食い違いがないかを点検する機能です。

1. **『フォーマット変更チェック』画面を開く** — 管理ブックの下のタブから選びます。
2. **対象のフォーマットを選ぶ** — 点検したい型をドロップダウンから選びます。
3. **『チェック』ボタンを押す** — 全ナレッジを調べ、項目の過不足や型の食い違いを一覧表示します。

!!! tip "結果の見方"
    何も表示されなければ問題なし。食い違いがあれば、登録画面から個別に直してください。

## 保存先フォルダを設定する

1. **『格納先設定』画面を開く** — 管理ブックの下のタブから選びます。4 つのフォルダの現在地が表示されます。
2. **変えたいフォルダにチェックを入れる** — 行の左のチェックボックスにチェック（複数同時も可）。
3. **『格納先を選ぶ』で新しい場所を指定** — フォルダ選択ダイアログで新しい場所を選び OK。
4. **『保存』ボタンを押す** — 次回起動から新しい保存先を参照します。

!!! warning "注意"
    保存先を変えても既存のファイルは自動で移動しません。先にエクスプローラーで中身をコピーしてから保存してください。

## 全体の設定とログを見る

### ⚙ 設定画面

- 表示色や既定フォーマットなど、全体の設定を変更できる
- 変更後は Excel を再起動すると反映される
- ふだんは初期設定のままで問題ない

### 📋 ログ画面

- 登録・修正・削除・検索の操作履歴を日時順に表示
- いつ・どの操作・どのナレッジ・成功か失敗かが分かる
- うまく動かないときの原因調べにも役立つ
