# テスト標準フォーマット (FMT-001.txt)

**配置先**: `C:\KnowledgeMgr\formats\FMT-001.txt`

メモ帳で **ANSI** (Shift-JIS) として保存してください。改行は CRLF。

```text
[FORMAT]
FormatId=FMT-001
FormatName=テスト標準フォーマット
===
[FIELD]
FieldName=件名
FieldType=単一行
Required=TRUE
MaxLength=120
fieldPlaceholder=（例）顧客対応マニュアル
searchTarget=TRUE
===
[FIELD]
FieldName=内容
FieldType=複数行
Required=TRUE
Rows=5
Scroll=TRUE
fieldPlaceholder=（例）対応の流れや手順を記入してください
searchTarget=TRUE
===
[FIELD]
FieldName=備考
FieldType=複数行
Required=FALSE
Rows=3
Scroll=FALSE
fieldPlaceholder=（例）注意事項や補足情報
searchTarget=TRUE
===
[FIELD]
FieldName=作成日
FieldType=日時
Required=FALSE
fieldPlaceholder=（例）2026-06-09
searchTarget=FALSE
===
[FIELD]
FieldName=カテゴリ
FieldType=選択
Required=FALSE
DropdownOptions=重要度1|重要度2|重要度3
fieldPlaceholder=（例）重要度2
searchTarget=FALSE
===
```
