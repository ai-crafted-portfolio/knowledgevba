# SAGYO-0001 - サーバ室UPS月次点検

**配置先**: `C:\KnowledgeMgr\data\SAGYO-0001.txt`

メモ帳で **ANSI** (Shift-JIS) として保存してください。改行は CRLF。

```text
###formatId###
SAGYO
###手順書番号###
OPS-001
###作業名###
サーバ室UPS定期点検
###対象設備###
サーバ室1F UPS本体（APC Smart-UPS 3000）
###作業区分###
定期点検
###危険度###
中
###作業予定日時###
2026-07-15 22:00
###必要な資格###
電気主任技術者第三種
###作業手順###
1. 事前にシステム部全員へ作業通知メールを送信
2. UPSの自己診断LEDの状態を目視確認
3. バッテリ電圧をテスタで計測（13.8V±0.2V以内であること）
4. UPSの動作テストモードを実行
5. テスト結果を点検表に記録
6. 点検完了報告メールをシステム部全員へ送信
###使用する工具・部品###
デジタルテスタ、点検表、絶縁手袋
###安全上の注意###
必ず2名以上で作業すること。
感電防止のため絶縁手袋を着用すること。
点検中は他のスタッフが誤って機器に触れないよう、入口にカラーコーンを設置する。
###確認者###
山田 太郎
```
