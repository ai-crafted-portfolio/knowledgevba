###概要###
社内メールの送信遅延が発生。通常2分以内に送信完了するが30分以上待たされる事象が複数発生。受信ログを確認すると一部のメッセージのみが影響を受けていた。
###原因###
メールサーバのキューが詰まっており、特定の宛先ドメインへの送信が滞っていた。SMTP接続プールの設定が小さすぎたことが原因。プール数を増やして解消。
