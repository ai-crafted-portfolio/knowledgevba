# サンプルデータ

knowledgevba を試すためのサンプルデータです。フォーマット定義ファイルとナレッジデータファイルを掲載しています。

## 配置先

| 種類 | 配置先フォルダ |
|---|---|
| フォーマット定義 | `C:\KnowledgeMgr\formats\` |
| ナレッジデータ | `C:\KnowledgeMgr\data\` |

各ファイルの内容をコピーして、メモ帳で **ANSI** (Shift-JIS) として保存してください。改行は CRLF。
