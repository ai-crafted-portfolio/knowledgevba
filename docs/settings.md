# 設定値の変更

ナレッジ管理ツール v2.3 の動作は、ブックごとの **「格納先設定」シート** で調整します。このページでは、利用者が変更する設定値とその変更方法を説明します。

## 1. 格納先設定シート

設定は、各ブック（検索.xlsm / 管理.xlsm）の中の **「格納先設定」シート** に 9 項目としてまとまっています。以前のテキスト形式の設定ファイルは v2.3 で廃止しました。設定はブックと一緒に保存されます。

シートは 6 つの保存先フォルダ（`data_dir` / `format_dir` / `ui_dir` / `log_dir` / `backup_dir` / `config_dir`）と、3 つの動作設定（`debugLevel` / `logRotationRows` / `uiSchemaFailMode`）で構成されます。

### 変更のしかた

1. **管理.xlsm の M-10「格納先設定」画面** を開きます（検索.xlsm も同じ「格納先設定」シートを持ちます）。
2. 値の列のセルを書き換えます。`debugLevel` と `uiSchemaFailMode` はドロップダウンから選べます。
3. **[保存]** ボタンを押します。ブックが上書き保存され、設定がブックに保持されます。
4. 変更は次回起動時から有効になります。保存先フォルダを変えたときは、ブックを開き直してください。

!!! note "初回起動時の自動設定"
    インストール後にブックを初めて開いたとき、6 つの保存先がすべて空であれば、保存先の親フォルダをたずねるダイアログが表示されます（既定 `C:\KnowledgeMgr`）。入力すると、その下に 6 つのフォルダが自動で設定され、ブックに保存されます。設定し忘れても既定値で起動します。

## 2. 格納先フォルダ

ナレッジ・フォーマット・画面定義・バックアップを置くフォルダを指定します。いずれも絶対パスで、末尾に `\` を付けます。

| 設定キー | 内容 |
|---|---|
| `data_dir` | ナレッジ本体（`<ナレッジ番号>.txt`）の置き場所 |
| `format_dir` | フォーマット定義（`<フォーマット ID>.txt`）の置き場所 |
| `ui_dir` | 画面レイアウト定義（`<ブック>\<画面>.txt`）の置き場所 |
| `backup_dir` | フォーマット変更時などに取得するバックアップの置き場所 |
| `log_dir` | 操作ログの置き場所 |
| `config_dir` | 設定の基準フォルダ |

格納先フォルダは、管理.xlsm の **M-10 格納先設定** 画面で変更し、**[保存]** ボタンで保存します。フォルダを別の場所へ移したときは、ファイルを移動したうえでこれらのパスを合わせてください。

## 3. ログの詳細度（debugLevel）

`debugLevel` は、LOG シートに残す操作ログの詳しさを決めます。値は次の 6 段階です。上にいくほど記録量が少なく、下にいくほど詳しくなります。

| 値 | 記録される内容 |
|---|---|
| `OFF` | 記録しない |
| `ERROR` | エラーのみ |
| `WARN` | エラーと警告 |
| `INFO` | 上記に加えて通常の操作 |
| `DEBUG` | 上記に加えて開発・調査向けの詳細 |
| `TRACE` | 最も詳しい記録 |

`debugLevel` の初期値は `INFO` です。通常の運用では `INFO` のままで十分です。不具合を調べたいときだけ一時的に `DEBUG` や `TRACE` に上げ、調査が終わったら戻すと、LOG シートが余計に大きくなりません。

`debugLevel` は、管理.xlsm の **M-11 設定**画面からも変更できます。

関連する設定として、`logRotationRows` は LOG シートに保持するログ行数の上限です（既定 10000 行）。上限に達すると古い行から自動で削除されます。

## 4. その他の動作設定

| 設定キー | 内容 | 既定値 |
|---|---|---|
| `uiSchemaFailMode` | 画面レイアウト定義に不備があったときの挙動。`safeDefault` はその画面だけエラー表示にして他の画面は通常起動、`abort` は 1 件でも不備があれば全画面の起動を中止 | `safeDefault` |
| `logRotationRows` | LOG シートに保持するログ行数の上限。超えると古い行から削除 | `10000` |

## 5. ファイル形式設定（M-13）

出力するファイルの形式に関する設定は、管理.xlsm の **M-13 ファイル形式設定**画面でまとめて行います。M-13 には文字コード・改行・テンプレートなどに関する 15 項目があり、画面上で値を選んで設定します。

M-13 の画面には、設定の不整合などを知らせるエラー表示帯（4 列の表）が連動して表示されます。

## 6. 画面の見た目を変えたいとき

ボタンの位置・ラベル・色、入力枠の大きさ、列幅といった画面の見た目は、格納先設定ではなく画面レイアウト定義ファイル（`ui\<ブック>\<画面>.txt`）で決まります。変更方法は[画面のカスタマイズ](screen_customization_guide_v2.md)を参照してください。
