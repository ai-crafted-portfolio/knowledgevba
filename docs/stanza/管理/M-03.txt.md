# M-03 フォーマット設計

`フォーマット設計` 画面の設定ファイルです。

## 配置先

以下のパスに **そのままのファイル名で** 配置してください。

```text
C:\KnowledgeMgr\ui\管理\M-03.txt
```

## 保存手順

1. 下の **内容** を全選択してコピー (右上のコピーボタンが便利)
2. **メモ帳** を開いて貼り付け
3. 「ファイル」 → 「名前を付けて保存」
4. ファイル名を `M-03.txt` にする
5. **文字コードを「ANSI」(Shift_JIS) に変更** ← 重要
6. 改行コードは「CRLF」(Windows 標準) のままにする
7. 上記 **配置先** のパスに保存

> 文字コードを UTF-8 のままにすると、`管理.xlsm` 等を開いたときに
> 画面タイトルや項目名が **文字化け** します。必ず `ANSI` で保存してください。
