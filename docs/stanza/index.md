# 画面設定ファイル一覧

ナレッジ管理ツールの各画面は、シート名・列幅・色・項目名などを **設定ファイル** で外出ししています。
設定ファイルを差し替えるだけで画面の見た目を変更できます。

## 配置先

設定ファイルは、ロール (管理 / 登録修正 / 検索) ごとに以下のフォルダに置きます。

```text
C:\KnowledgeMgr\ui\管理\
C:\KnowledgeMgr\ui\登録修正\
C:\KnowledgeMgr\ui\検索\
```

各 `M-XX.txt` を該当ロールのフォルダに **そのままのファイル名で** 配置してください。

## 保存時の注意

各設定ファイルは **メモ帳で「ANSI」(Shift_JIS) 文字コードで保存** してください。
UTF-8 のまま保存すると、画面タイトルや項目名が文字化けします。
詳しい手順は各ファイルのページに書かれています。
