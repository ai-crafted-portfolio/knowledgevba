# ナレッジ管理ツール (VBA 製)

障害対応の記録・手順書・ノウハウなどを、Excel から登録・検索・編集できる内製ツールです。データはすべて Excel ブックの外側にテキストファイルとして保存し、Excel ブックは「データを表示・編集するための窓口」として動きます。

このサイトは、現行バージョン **v2.1** の構成・導入手順・操作手順・設計をまとめた公開ドキュメントです。

!!! tip "このドキュメントで使う用語"
    - **VBA** — Excel に標準搭載されているプログラミング言語（Visual Basic for Applications）です。
    - **VBE** — VBA のコードを編集するエディタ（Visual Basic Editor）。Excel 上で `Alt + F11` で開きます。
    - **`.xlsm`** — マクロ（VBA）を含められる Excel ブックの形式です。
    - **スタンザ** — `[セクション]` 行と `key=value` 行の組み合わせで書かれたテキスト形式です。本ツールは画面・フォーマット・設定・ナレッジの定義をこの形式のテキストファイルで持ちます。

## このツールでできること

過去の対応記録やノウハウを、フォーマット（入力項目のひな型）を決めて整理し、後からキーワードや条件で探し出せます。記録はテキストファイル 1 件 = 1 ナレッジで保存されるため、Excel を使わずテキストエディタで中身を確認することもできます。入力項目は固定ではなく、業務に合わせてフォーマットを設計して増やせます。

## 3 つのブックで構成されています

v2.1 では、役割の異なる 3 つの Excel ブックに分かれています。利用者は自分の作業に対応するブックを開いて使います。

| ブック | 主な役割 | 含まれる画面 |
|---|---|---|
| 登録修正.xlsm | ナレッジの新規登録と修正 | M-05 ナレッジ登録 / M-06 ナレッジ修正 |
| 検索.xlsm | ナレッジの一覧・検索・閲覧 | M-07 ナレッジ一覧 / M-08 ナレッジ検索 / M-09 ナレッジ表示 |
| 管理.xlsm | フォーマット設計・各種設定・操作ログ | M-02〜M-04 / M-10〜M-14 の 8 画面 |

画面は **M-02 から M-14 までの 13 画面**です。各画面は Excel のシートタブで切り替えます。

## データの持ち方

すべての永続データは Excel ブックの外側のフォルダ（既定では `C:\KnowledgeMgr\`）にテキストファイルとして置かれます。

- **ナレッジ本体** — 1 件 1 ファイル（`<ナレッジ番号>.txt`）
- **フォーマット定義** — 入力項目のひな型（`<フォーマット ID>.txt`）
- **画面レイアウト定義** — 各画面の見た目を決めるテキスト
- **動作設定** — フォルダの場所やログの詳細度などを書いた設定ファイル（ブックごとに 1 つ）

Excel ブック側はこれらを読み込んで画面を組み立て、表示・入力を仲介します。文字コードは Shift_JIS、改行は CRLF で統一しています。

## 設計のポイント

- **3 ブック構成** — 登録・検索・管理を別ブックに分け、それぞれの利用者が必要な機能だけを開けるようにしています。
- **外部スタンザ駆動** — 画面レイアウトもフォーマットも Excel の外側のテキストで定義します。`.xlsm` の中のコードを書き換えずに、項目や見た目を変更できます。
- **層分離** — VBA モジュールを役割ごとの層に分け、外部ファイルの読み書きは専用のモジュールにまとめています。
- **モジュール配布方式** — 完成した `.xlsm` を配るのではなく、VBA モジュールを取り込んでセットアップする形で各 PC に展開します。
